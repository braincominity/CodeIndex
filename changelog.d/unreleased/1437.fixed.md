## English

- **Exact graph lookups use composite reference indexes (#1437)** — `references`, `callers`, and `callees` now create composite `symbol_references` indexes for case-insensitive and Unicode-folded exact lookups paired with reference kind or source file filtering, reducing per-row filtering on high-fan-in symbols.

## 日本語

- **exact な graph lookup が複合 reference index を使うようになりました (#1437)** — `references` / `callers` / `callees` は、大文字小文字無視および Unicode 折り畳み済みの exact lookup で reference kind や source file の絞り込みと組み合わせられる `symbol_references` 複合 index を作成し、高 fan-in シンボルでの行単位フィルタを減らします。
