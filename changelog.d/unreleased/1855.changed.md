## English

- **Human CLI file-size output now uses binary units (#1855)** — `files` and `map` text output render sizes as `KiB` / `MiB` / `GiB` while `--bytes` preserves raw byte counts and `--json` keeps raw integer bytes for machines.

## 日本語

- **人間向け CLI のファイルサイズ出力を2進単位にしました (#1855)** — `files` と `map` のテキスト出力は `KiB` / `MiB` / `GiB` でサイズを表示し、`--bytes` では従来通り raw byte count を維持し、`--json` は機械処理向けに raw integer bytes のまま返します。
