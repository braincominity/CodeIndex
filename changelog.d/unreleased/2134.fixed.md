- **Suggestion records now persist GitHub submission attempt diagnostics (#2134)** — local suggestion JSON now records the last submit attempt time, attempt count, and last submission error so operators can distinguish never-attempted, retryable, and rejected suggestions.

## 日本語

- **Suggestion record が GitHub 送信試行の診断情報を永続化するようになりました (#2134)** — ローカルの suggestion JSON に最後の送信試行時刻、試行回数、最後の送信エラーを記録し、未試行・再試行可能・拒否済みの提案を運用者が区別できるようにしました。
