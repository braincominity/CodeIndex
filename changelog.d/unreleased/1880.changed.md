## English

- **Call-graph reference kinds now use a consistent logical taxonomy (#1880)** — default `callers` / `callees` output reports `call` and `instantiate` as `invoke`, event subscription rows as `event`, and adds `--raw-kinds` for users who need raw extractor labels.

## 日本語

- **call-graph の reference kind が一貫した logical taxonomy を使うようになりました (#1880)** — 既定の `callers` / `callees` 出力では `call` と `instantiate` を `invoke`、イベント購読行を `event` として返し、extractor の raw label が必要な場合に使う `--raw-kinds` を追加しました。
