## English

- **C# static member qualifiers are now indexed as class references (#1404)** — `Class.Member(...)` and `Class.Field` usages now create reference edges to the class qualifier, so `references`, `callers`, `impact`, and `unused` no longer miss classes used only through static member access.

## 日本語

- **C# の static member qualifier をクラス参照として索引するようになりました (#1404)** — `Class.Member(...)` や `Class.Field` の使用箇所がクラス qualifier への参照 edge を作るため、static member access 経由でしか使われないクラスを `references`、`callers`、`impact`、`unused` が見落とさなくなりました。
