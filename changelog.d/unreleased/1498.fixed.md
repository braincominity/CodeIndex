## English

- **CSS reference extractor now emits `#id` selector references (#1498)** — `body #header { ... }` now produces a `reference` edge to `#header`, so `callers` / `impact` follow CSS ID selectors instead of treating defined IDs as unused. Hex color literals such as `#fff` and `#abc123` in property values are still ignored.

## 日本語

- **CSS の参照抽出が `#id` セレクタ参照を発行するようになりました (#1498)** — `body #header { ... }` が `#header` への `reference` エッジを生成するようになり、`callers` / `impact` で CSS の ID セレクタを追跡できます。プロパティ値中の `#fff` や `#abc123` などの hex color リテラルは引き続き無視されます。
