## English

- **C# `unused` now recognizes direct reflection member-name literals (#1861)** - `GetMethod("Foo")` and literal-concat forms such as `GetProperty("Display" + "Name")` now emit indexed references, reducing false unused results for reflection-heavy code while still leaving dynamically constructed names for manual review.

## 日本語

- **C# の `unused` が直接の reflection member-name literal を認識するようになりました (#1861)** - `GetMethod("Foo")` や `GetProperty("Display" + "Name")` のような literal 連結が参照として index され、reflection を多用するコードでの未使用誤検出を減らします。動的に組み立てられる名前は引き続き手動確認対象です。
