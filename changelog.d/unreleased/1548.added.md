## English

- **Golden-file regression fixtures for the CLI `--json` output contracts (#1548)** — added `JsonOutputSnapshotTests` plus a `JsonOutputSnapshotHelper` that runs `status` / `search` / `references` / `impact` / `excerpt` against deterministic in-memory fixtures, normalizes volatile fields (timestamps, absolute paths, commit SHAs, FTS5 scores, version strings), and diffs the result against checked-in golden files under `tests/CodeIndex.Tests/golden/`. Renames, removals, reordered arrays, or new keys now fail the snapshot, so any `--json` shape change is forced to land alongside an intentional golden update. The update procedure (set `UPDATE_SNAPSHOTS=1` and re-run only the snapshot tests, then review the diff) is documented in `TESTING_GUIDE.md`.
