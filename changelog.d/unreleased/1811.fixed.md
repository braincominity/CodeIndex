## English

- **MCP tools now advertise authoritative language support (#1811)** — `tools/list` descriptions now include a `Language support:` clause generated from the same detected, symbol, and graph language registries used at runtime, so clients no longer need trial-and-error to understand each tool's language scope.

## 日本語

- **MCP ツールが権威ある言語サポート範囲を明示するようになりました (#1811)** — `tools/list` の説明に、実行時と同じ検出・シンボル・グラフ言語レジストリから生成した `Language support:` 句を含めるようにし、クライアントが各ツールの言語範囲を試行錯誤せず判断できるようにしました。
