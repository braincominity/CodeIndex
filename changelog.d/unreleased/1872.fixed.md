## English

- **`outline` now orders same-line symbols deterministically (#1872)** - symbol start columns are persisted when available, and `outline` now orders rows by line, start column, kind, and name so repeated outline diffs no longer depend on SQLite row order.

## 日本語

- **`outline` が同一行シンボルを決定的に並べるようになりました (#1872)** - 利用可能なシンボル開始列を永続化し、`outline` は行、開始列、種別、名前の順に並べるため、繰り返し取得したアウトライン差分が SQLite の行順に左右されなくなりました。
