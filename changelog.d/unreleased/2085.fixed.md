- **C# async iterators now expose implicit iterator dependencies (#2085)** — `IAsyncEnumerable<T>` methods and local functions now emit type references for async iterator interfaces and generic arguments, plus `implicit_implementation` references for `GetAsyncEnumerator` and `MoveNextAsync`.
