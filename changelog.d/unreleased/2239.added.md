## English

- **Branch-switch partial indexing can now target changed refs (#2239)** — `cdidx index <projectPath> --changed-between <old-ref> <new-ref>` updates only files changed between two git refs and includes both old and new rename paths so stale indexed paths can be purged.

## 日本語

- **ブランチ切り替え向けの ref 間部分更新を追加しました (#2239)** — `cdidx index <projectPath> --changed-between <old-ref> <new-ref>` は2つの git ref 間で変更されたファイルのみ更新し、rename の旧パスと新パスを両方含めるため古い indexed path も purge できます。
