## English

- **Deprecated `find --count --json` `file_count` alias (#1423)** - `files` is now documented and tested as the canonical matched-file count, while `file_count` remains as a compatibility alias with the same value for the current major release.

## 日本語

- **`find --count --json` の `file_count` エイリアスを非推奨にしました (#1423)** - `files` を一致ファイル数の正規フィールドとして文書化しテストで固定しました。`file_count` は現在のメジャーリリース中、同じ値を返す互換エイリアスとして残ります。
