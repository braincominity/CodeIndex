- **C# default interface method generic constraints now keep method context (#2087)** — type references emitted from `where` clauses on braced or expression-bodied interface methods now attach to the owning method instead of falling out of scope.

- **C# の default interface method にある generic 制約がメソッド文脈を保つようになりました (#2087)** — braced / expression-bodied な interface method の `where` 句から出る型参照が、スコープ外になるのではなく所有メソッドに帰属するようになりました。
