## English

- **NuGet restore now uses a repository-local trust policy (#1676)** — `nuget.config` restricts restores to nuget.org, requires signed packages, and pins the NuGet.org repository signer so untrusted feeds and unsigned packages are rejected.

## 日本語

- **NuGet restore がリポジトリ内の信頼ポリシーを使うようになりました (#1676)** — `nuget.config` で restore 元を nuget.org に限定し、署名付きパッケージを必須化し、NuGet.org の repository signer を pin することで、未信頼フィードや未署名パッケージを拒否します。
