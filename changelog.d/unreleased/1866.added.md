## English

- **`hotspots` now exposes explicit grouping metadata and selection (#1866)** — `hotspots --group-by=symbol|file|statement` and MCP `symbol_hotspots` `groupBy` let callers choose the grouping unit, while JSON responses include `grouped_by`; SQL scopes keep their existing statement-oriented default and non-SQL scopes default to symbol grouping.

## 日本語

- **`hotspots` が明示的な grouping metadata と選択肢を公開するようになりました (#1866)** — `hotspots --group-by=symbol|file|statement` と MCP `symbol_hotspots` の `groupBy` で集計単位を選べるようになり、JSON 応答には `grouped_by` が含まれます。SQL scope は既存の statement-oriented 既定を維持し、非 SQL scope は symbol grouping を既定にします。
